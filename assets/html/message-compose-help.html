This is the help text for the message composer tab . The message composer allows you to build messages consisting of phrases and text. You build the messages either by tapping on suggested phrases (i.e. an auto-suggest), or by directly typing in text.
Scroll through to read this help. Tap to close the help pane at any point, and tap to bring it back. In general, tap to switch between open/closed.
The composer is split into three main regions: the Input text (top), the Suggestions (middle), and the Expanded text (bottom). If you are logged in (via the settings tab ), you may also choose whether you want a Public or a Direct message, and if the latter, you will need to specify a recipient.
The Input text can include regular text as well as phrases. Phrase names can include alphanumeric characters (case-insensitive) or underscores, and must be preceded by the 'PHRASE' symbol, like this: PHRASEan_example_phrase
To add to the Input text, tap one of the phrases in the Suggestions region. To delete a phrase added in this way, tap the which appears in Suggestions. Alternatively, tap to edit the Input text manually with the keyboard, or just tap the Input text itself (but be careful not to tap a phrase link in the Input text, as this will take you to the definition for that phrase in the phrasebook tab ).
When you change the Input text, the Expanded text is automatically (and randomly) updated using the definitions in the phrase book. The update of the Expanded text is a random generative process, and may not come out the same way twice. If you don't happen to like how it came out, you can at any time tap for another roll of the dice to re-randomize the Expanded text.
When you are satisfied with the Expanded text, tap to post it. You can share using Twitter or Facebook , or just on this website as a public message , or (if you are logged in) a direct message to another player's mailbox . Alternatively, if you change your mind about sharing, close the share panel by tapping the close icon or by just tapping again.
