This is the help text for the message composer tab . The message composer allows you to build messages consisting of text and synonym expansions from the thesaurus. You build the messages either by tapping on one of the auto-suggestions, or by directly typing in text.
Scroll through to read this help. Tap to close the help pane at any point, and tap to bring it back. In general, tap to switch between open/closed.
By default, the view above shows a message card. Swipe left or tap to throw away this card and deal a new card. Swipe right or tap to post the message online.
You can open up the composer by tapping and close it again by tapping again.
The composer is split into three main regions: the Template text (top), the Suggestions (middle), and the message card itself (bottom). If you are logged in (via the settings tab ), you may also choose whether you want a Public or a Direct message, and if the latter, you will need to specify a recipient.
The Template text can include regular text as well as synonym expansions, which must be preceded by the 'PHRASE' symbol, like this: PHRASEan_example_phrase This will look up 'an_example_phrase' in the thesaurus, and (if an entry for that phrase exists) it will randomly select one of the synonyms defined for that phrase, and insert that random definition into the Message text. Phrase names can include alphanumeric characters (case-insensitive) or underscores.
To add to the Template text, tap one of the phrases in the Suggestions region. To delete a phrase added in this way, tap the which appears in Suggestions. Alternatively, tap the Template text window to edit the template manually with the keyboard (be careful not to tap a synonym-expansion link in the Template text, as this will take you to the definition for the corresponding phrase in the thesaurus tab ).
When you change the Template text, the Message text is automatically (and randomly) updated using the definitions in the thesaurus. The generation of the Message text from the Template text is a random process; that is, it uses random (and potentially different) synonyms each time, so it may not come out the same way twice in a row. If you don't happen to like how it came out, you can at any time swipe left or tap for another shuffle of the deck to re-randomize the Message text using different synonyms from the thesaurus.
When you are satisfied with the Message text, swipe right or tap to post it. You can share using Twitter or Facebook , or just on this website as a public message , or (if you are logged in) a direct message to another player's mailbox . Alternatively, you can just copy the message to the clipboard or, if you change your mind about sharing, you can close the share panel by tapping the close icon or by just tapping again.
